  ÉTAPE 1 : Pousser votre code sur GitHub (3 minutes)
  ────────────────────────────────────────────────────

  Ouvrez PowerShell dans ce dossier et tapez :

    git init
    git add .
    git commit -m "Deploy to Render"
    
    # Créez un nouveau repo sur https://github.com/new
    # Puis remplacez USERNAME par votre nom d'utilisateur :
    
    git remote add origin https://github.com/USERNAME/linguameet.git
    git branch -M main
    git push -u origin main


  ÉTAPE 2 : Créer un compte Render (2 minutes)
  ─────────────────────────────────────────────

  1. Visitez : https://render.com
  2. Cliquez sur "Get Started"
  3. Inscrivez-vous avec GitHub (recommandé)


  ÉTAPE 3 : Déployer sur Render (5 minutes)
  ──────────────────────────────────────────

  1. Allez sur : https://dashboard.render.com
  2. Cliquez sur "New +" → "Blueprint"
  3. Connectez votre repository GitHub "linguameet"
  4. Render détecte automatiquement render.yaml
  5. Cliquez sur "Apply"
  6. Attendez 5-10 minutes


  ÉTAPE 4 : Votre app est en ligne ! ✨
  ─────────────────────────────────────

  Vous obtiendrez une URL du type :
    https://linguameet.onrender.com
